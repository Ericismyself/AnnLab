知识图谱构建

本演示展示了从非结构化文本中自动抽取实体、关系，构建领域知识图谱的完整流程。 以奶业金融供应链场景为例，预置伊利集团相关文本片段。

基于知识图谱的知识补全

展示知识图谱补全算法，对缺失的实体和关系进行预测与推理。 以医疗领域（糖尿病）知识图谱为例，演示推理过程与置信度计算。

知识融合与推理

将"马蜂窝风格"和"大众点评风格"的内蒙古旅游知识图谱进行对齐融合演示。 三步式交互：实体对齐 → 执行融合 → 完成推理。

多模态知识图谱

融合文本、图像、知识的多模态知识图谱展示，涵盖奶业金融、医疗、旅游三大领域。 点击任意节点查看图文详情与邻居关系。

负载建模 赛博风格

容器云环境下的工作负载特征分析与建模，支持 CPU / 内存 / 网络 I/O / QPS / 磁盘 I/O 五类指标的实时监控与类型识别。

负载预测 赛博风格

基于 LSTM 深度学习模型的时间序列预测，灰色区域为历史数据，粉色区域为预测结果及置信区间。 可切换 1h / 6h / 24h 预测范围。

自动弹性扩缩 赛博风格

基于预测的自动化容器扩缩容策略演示，点击"INJECT LOAD"按钮观察扩缩容过程。 Matrix 风格背景 + 实时容器可视化。

异常检测 赛博风格

容器云环境下的资源异常实时检测与告警，点击按钮触发各类异常事件观测。 支持 CPU 飙升、内存泄漏、网络异常、抖动异常四类场景。
